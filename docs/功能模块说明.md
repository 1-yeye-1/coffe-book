# 功能模块说明

## 前台用户端

| 模块 | 主要功能 | 说明 |
| --- | --- | --- |
| 首页 | 品牌展示、核心入口、活动与商品摘要 | 作为项目第一屏，展示咖啡书屋定位和业务入口。 |
| 咖啡文化 | 咖啡饮品展示、加入购物车 | 与商城共用购物车和订单流程。 |
| 精品书库 | 图书列表、图书详情 | 展示馆藏、作者、出版社和简介。 |
| 文创商城 | 商品浏览、分类、商品详情、数量选择、加入购物车 | 商品卡片和详情页均支持库存、价格和交互反馈。 |
| 购物车 | 数量修改、删除、全选、结算 | 登录用户可进入，操作会同步后端购物车。 |
| 订单与支付 | 创建订单、订单列表、订单详情、取消、支付页面 | 支付后可由后台审核状态，取消未支付订单会恢复库存。 |
| 在线预约 | 日期时间选择、座位图、多人预约、备注 | 根据日期和时段显示座位状态。 |
| 我的预约 | 查看和取消个人预约记录 | 只展示当前登录用户的预约，取消时校验归属。 |
| 活动报名 | 活动列表、详情、报名人数、会员权益、活动状态 | 支持普通报名、提前报名、满员和重复报名拦截。 |
| 书友社区 | 发帖、评论、点赞、用户主页 | 登录后可发布内容，后台可审核评论。 |
| 会员中心 | 个人资料、积分、礼券、收藏、笔记、通知 | 登录后通过右上角账户菜单进入。 |

## 后台管理端

| 模块 | 主要功能 | 说明 |
| --- | --- | --- |
| 管理员登录 | 管理员账号密码登录 | 使用独立管理员 token，普通用户 token 不能访问后台接口。 |
| 工作台 | 指标卡片、实时日志摘要 | 用于答辩展示系统运营状态。 |
| 实时日志 | 操作记录查询 | 记录用户、管理员和系统行为。 |
| 用户管理 | 新增、编辑、删除、搜索、筛选、分页 | 支持维护会员资料和积分。 |
| 商品管理 | 新增、编辑、删除商品 | 删除操作有确认弹窗。 |
| 图书管理 | 维护图书展示数据 | 支持后台补充馆藏内容。 |
| 订单管理 | 查看订单、审核支付、搜索分页 | 支持订单支付审核流程。 |
| 预约管理 | 查看、修改、删除预约 | 可处理座位使用状态。 |
| 活动管理 | 活动维护、状态管理与报名记录查看 | 支持容量、时间、报名规则和草稿/开放/关闭状态。 |
| 社区审核 | 帖子管理、评论审核 | 支持评论通过、驳回和删除。 |
| 内容管理 | 公告维护 | 用于运营公告和首页内容补充。 |
| 收入查看 | 支付订单收入统计 | 可展示订单数和收入总额。 |
| 数据库查看 | 表结构和记录数 | 便于说明数据库设计和模块对应关系。 |
| 数据看板 | 指标汇总 | 适合毕业设计展示。 |

## 公共能力

- 统一 API 请求封装，自动附带用户或管理员 token。
- Vue Router 登录态守卫，未登录访问会员、购物车、订单等页面会跳转到登录页。
- 表单校验覆盖手机号、密码、数量、文本长度和必填项。
- 空状态、加载状态、错误状态统一展示，减少页面突兀感。
- 后台表格统一搜索、筛选、分页和操作提示。

## 会员积分模块

- 会员等级包含普通会员、黄金会员、钻石会员。
- 签到和订单支付审核通过后会增加积分与等级成长值。
- 积分中心支持兑换咖啡券、文创优惠券和活动权益。
- 礼券记录保存在会员数据中，使用后会更新状态。

## 商城订单模块

- 商品支持分类、关键词筛选、库存、价格、图片展示和详情页查看。
- 购物车支持数量修改、删除、全选、结算和空状态提示，登录用户同步后端购物车。
- 订单创建时会校验商品库存和购买数量，创建成功后扣减库存。
- 订单列表和订单详情只返回当前登录用户数据，取消未支付订单时恢复库存。
- 支付提交后进入后台审核流程，审核通过后订单变为已支付。

## 在线预约模块

- 用户可按日期和时间段查看座位状态。
- 预约时校验手机号、人数、座位数量和座位冲突。
- 我的预约只展示当前登录用户的数据。
- 取消预约时服务端校验预约归属，防止操作他人预约。
- 后台新增或编辑预约时也会校验同日同时间段座位冲突。

## 书友社区模块

- 用户登录后可发布帖子、评论和点赞。
- 评论默认进入待审核状态，后台可通过、驳回或删除。
- 用户主页受隐私设置控制，关闭公开展示后他人无法查看。

## 活动赛事模块

- 活动包含容量、日期、时间、地点、报名开放时间和提前报名时间。
- 用户报名时校验容量、重复报名和会员提前报名资格。
- 后台可维护活动信息、草稿/开放/关闭状态，并查看报名记录。
