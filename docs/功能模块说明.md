# 功能模块说明

## 前台用户端

| 模块 | 主要功能 | 说明 |
| --- | --- | --- |
| 首页 | 品牌展示、核心入口、活动与商品摘要 | 作为项目第一屏，展示咖啡书屋定位和业务入口。 |
| 咖啡文化 | 咖啡饮品展示、加入购物车 | 与商城共用购物车和订单流程。 |
| 精品书库 | 图书列表、图书详情 | 展示馆藏、作者、出版社和简介。 |
| 文创商城 | 商品浏览、分类、数量选择、加入购物车 | 商品卡片支持库存、价格和交互反馈。 |
| 购物车 | 数量修改、删除、结算 | 登录用户可进入，防止未登录访问。 |
| 订单与支付 | 创建订单、支付页面、支付结果 | 支付后可由后台审核状态。 |
| 在线预约 | 日期时间选择、座位图、多人预约、备注 | 根据日期和时段显示座位状态。 |
| 我的预约 | 查看个人预约记录 | 只展示当前登录用户的预约。 |
| 活动报名 | 活动详情、报名人数、会员权益 | 支持普通报名和提前报名规则。 |
| 书友社区 | 发帖、评论、点赞、用户主页 | 登录后可发布内容，后台可审核评论。 |
| 会员中心 | 个人资料、积分、礼券、收藏、笔记、通知 | 登录后通过右上角账户菜单进入。 |

## 后台管理端

| 模块 | 主要功能 | 说明 |
| --- | --- | --- |
| 管理员登录 | 管理员账号密码登录 | 使用独立管理员 token，普通用户 token 不能访问后台接口。 |
| 工作台 | 指标卡片、实时日志摘要 | 用于答辩展示系统运营状态。 |
| 实时日志 | 操作记录查询 | 记录用户、管理员和系统行为。 |
| 用户管理 | 新增、编辑、删除、搜索、筛选、分页 | 支持维护会员资料和积分。 |
| 商品管理 | 新增、编辑、删除商品 | 删除操作有确认弹窗。 |
| 图书管理 | 维护图书展示数据 | 支持后台补充馆藏内容。 |
| 订单管理 | 查看订单、审核支付、搜索分页 | 支持订单支付审核流程。 |
| 预约管理 | 查看、修改、删除预约 | 可处理座位使用状态。 |
| 活动管理 | 活动维护与报名记录查看 | 支持容量、时间和报名规则。 |
| 社区审核 | 帖子管理、评论审核 | 支持评论通过、驳回和删除。 |
| 内容管理 | 公告维护 | 用于运营公告和首页内容补充。 |
| 收入查看 | 支付订单收入统计 | 可展示订单数和收入总额。 |
| 数据库查看 | 表结构和记录数 | 便于说明数据库设计和模块对应关系。 |
| 数据看板 | 指标汇总 | 适合毕业设计展示。 |

## 公共能力

- 统一 API 请求封装，自动附带用户或管理员 token。
- Vue Router 登录态守卫，未登录访问会员、购物车、订单等页面会跳转到登录页。
- 表单校验覆盖手机号、密码、数量、文本长度和必填项。
- 空状态、加载状态、错误状态统一展示，减少页面突兀感。
- 后台表格统一搜索、筛选、分页和操作提示。
