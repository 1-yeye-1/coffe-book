# 功能模块说明

## 前台用户端

| 模块 | 说明 |
| --- | --- |
| 首页 | 展示品牌主视觉、核心入口、推荐内容和会员引导。 |
| 咖啡文化 | 展示咖啡知识、文化故事、门店体验和主题内容。 |
| 精品书库 | 展示书籍列表、分类、推荐和图书详情入口。 |
| 文创商城 | 展示商品列表、筛选、数量选择、加入购物车和商品详情。 |
| 在线预约 | 提供日期、时间、人数、座位选择和预约提交。 |
| 书友社区 | 展示社区动态、发帖、评论、点赞和用户互动。 |
| 活动赛事 | 展示活动列表、活动详情和报名流程。 |
| 购物车 | 管理购物车商品、数量、选择状态和结算入口。 |
| 我的订单 | 展示订单状态、订单详情、取消订单和支付入口。 |
| 我的收藏 | 展示用户收藏内容。 |
| 我的笔记 | 展示用户阅读笔记。 |
| 消息中心 | 展示系统通知和业务消息。 |
| 积分中心 | 展示积分余额、成长值、任务和兑换权益。 |
| 我的礼品 | 展示礼券、礼品状态、二维码和使用记录。 |
| 个人资料 | 管理昵称、手机号、邮箱、头像、偏好、地址和隐私信息。 |
| 安全设置 | 管理安全评分、密码、绑定信息、登录设备、登录记录和安全建议。 |

## 后台管理端

| 模块 | 说明 |
| --- | --- |
| 工作台 | 展示运营概览、核心指标、趋势和快捷入口。 |
| 实时日志 | 展示用户和管理员操作日志。 |
| 用户管理 | 管理会员信息、等级、积分和用户状态。 |
| 商品管理 | 管理文创商品、价格、库存、分类和上下架信息。 |
| 书籍管理 | 管理馆藏书籍、作者、分类、简介和推荐信息。 |
| 订单管理 | 查看订单、编辑订单状态和管理订单数据。 |
| 支付审核 | 审核用户提交的支付记录，确认或驳回收款。 |
| 预约管理 | 管理座位预约、时间段、人数和预约状态。 |
| 活动管理 | 管理活动、名额、报名时间和活动状态。 |
| 社区审核 | 管理社区帖子、评论审核和内容删除。 |
| 内容管理 | 管理公告、运营内容和首页展示信息。 |
| 财务中心 | 汇总收入、订单金额和支付数据。 |
| 运营驾驶舱 | 展示关键运营指标、图表和业务趋势。 |
| 权限管理 | 展示角色、权限组和管理员权限信息。 |
| 系统设置 | 展示站点配置、安全策略和运维建议。 |
| 数据库对应 | 展示业务模块与数据库表的对应关系。 |

## 权限说明

- 未登录用户可浏览公开页面。
- 购物车、订单、会员中心、个人资料、安全设置等页面需要普通用户登录。
- 后台管理端需要管理员登录。
- 用户 token 不能访问后台接口，管理员 token 不能替代普通用户完成前台会员操作。
