# UI重构报告

## 重构目标

本次 UI 重构目标是将“咖啡书屋平台”从基础课程项目界面提升为可用于毕业设计展示的完整产品界面。重构重点包括：

- 统一前台和后台视觉语言。
- 提升首页、商城、预约、社区、会员中心等核心页面的信息层级。
- 保持 API、Pinia、Router 和后端逻辑稳定。
- 优化 PC 与移动端响应式体验。
- 消除破图、横向滚动、明显溢出和不统一排版。

## 视觉方向

- 主题：咖啡书屋、阅读空间、城市文化、会员运营。
- 色彩：以咖啡棕、暖米白、金色强调色为主。
- 组件：统一卡片、按钮、状态标签、表单、列表、表格、抽屉、弹窗和空状态。
- 响应式：前台页面适配 PC 与移动端，后台以桌面管理体验为主，同时控制页面级横向滚动。
