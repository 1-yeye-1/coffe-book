发现项：

- 首页首屏背景图存在视觉溢出记录，但页面级横向滚动为 false，属于全屏视觉图的裁切表现。
- 后台“数据库对应”模块表格内容较宽，但外层页面级横向滚动为 false，属于数据表格内部宽内容展示。
- 静态扫描发现 `AdminModal.vue` 疑似未使用。
- 静态扫描发现部分重复 CSS 选择器，主要来自历史阶段追加式 UI 精修样式。
