# 接口说明

## 接口基础地址

本地开发默认基础地址：

```text
http://localhost:4173
```

统一返回格式：

```json
{
  "success": true,
  "message": "ok",
  "data": {}
}
```

失败返回格式：

```json
{
  "success": false,
  "message": "错误原因",
  "data": null
}
```

登录后请求需要带 token：

```text
Authorization: Bearer <token>
```

## 用户认证接口

| 方法 | 路径 | 参数 | 说明 |
| --- | --- | --- | --- |
| GET | `/api/auth/slider` | 无 | 获取滑块验证码 |
| POST | `/api/auth/sms-code` | `phone`、`sliderToken`、`sliderValue` | 发送短信验证码 |
| POST | `/api/auth/register` | `name`、`phone`、`password`、`smsCode` | 用户注册 |
| POST | `/api/auth/login` | `phone`、`password` | 密码登录 |
| POST | `/api/auth/sms-login` | `phone`、`smsCode` | 短信验证码登录 |

登录请求示例：

```json
{
  "phone": "13800000000",
  "password": "coffee123"
}
```

登录返回示例：

```json
{
  "success": true,
  "data": {
    "user": { "id": 1, "name": "城市读者", "phone": "13800000000" },
    "token": "jwt-token"
  }
}
```

## 订单接口

| 方法 | 路径 | 参数 | 说明 |
| --- | --- | --- | --- |
| POST | `/api/cart` | `productId`、`quantity` | 加入购物车，需登录 |
| POST | `/api/orders` | `items` | 创建订单，需登录 |
| GET | `/api/orders/:id/payment-status` | 订单 id | 查询本人订单支付状态 |
| POST | `/api/orders/:id/pay` | `paymentMethod` | 提交本人订单支付 |

创建订单请求示例：

```json
{
  "items": [
    { "productId": 1, "quantity": 2 }
  ]
}
```

安全规则：

- 未登录不能创建订单。
- 用户只能查询和支付自己的订单。
- 后端会校验商品是否存在、库存是否足够、数量是否为整数。

## 预约接口

| 方法 | 路径 | 参数 | 说明 |
| --- | --- | --- | --- |
| GET | `/api/seats/status?date=YYYY-MM-DD&time=HH:mm` | `date`、`time` | 查询座位状态 |
| POST | `/api/reservations` | `phone`、`date`、`time`、`people`、`seatIds`、`purpose`、`note` | 创建预约 |
| DELETE | `/api/reservations/:id` | 预约 id | 取消本人预约 |

预约请求示例：

```json
{
  "phone": "13800000000",
  "date": "2026-06-03",
  "time": "14:00",
  "people": 1,
  "seatIds": ["A1"],
  "purpose": "阅读自习",
  "note": "靠窗"
}
```

安全规则：

- 我的预约通过 `/api/member` 返回，只包含当前登录用户的预约。
- 取消预约时会校验 `reservation.userId === 当前用户 id`。

## 社区接口

| --- | --- | --- | --- |
| GET | `/api/posts` | 无 | 查询公开帖子 |

## 活动接口

| 方法 | 路径 | 参数 | 说明 |
| --- | --- | --- | --- |
| GET | `/api/activities` | 无 | 查询活动列表 |
| GET | `/api/activities/:id` | 活动 id | 查询活动详情 |
| POST | `/api/activities/:id/apply` | `people`、`phone`、`kind` | 活动报名 |

活动报名会校验活动容量、重复报名、提前报名资格和每月提前报名次数。

## 后台管理接口

后台除登录外均需要管理员 token。普通用户 token 访问后台接口会返回 401。

| 方法 | 路径 | 参数 | 说明 |
| --- | --- | --- | --- |
| POST | `/api/admin/login` | `account`、`password` | 管理员登录 |
| GET | `/api/admin/summary` | 无 | 后台汇总数据 |
| GET | `/api/admin/realtime` | `page`、`pageSize` | 实时日志列表 |
| GET | `/api/admin/realtime/:id` | 日志 id | 日志详情 |
| POST/PATCH/DELETE | `/api/admin/users/:id?` | 用户字段 | 用户管理 |
| POST/PATCH/DELETE | `/api/admin/products/:id?` | 商品字段 | 商品管理 |
| POST/PATCH/DELETE | `/api/admin/books/:id?` | 图书字段 | 图书管理 |
| POST/PATCH/DELETE | `/api/admin/orders/:id?` | 订单字段 | 订单管理 |
| PATCH | `/api/admin/orders/:id/payment-review` | `status=approved/rejected` | 支付审核 |
| POST/PATCH/DELETE | `/api/admin/reservations/:id?` | 预约字段 | 预约管理 |
| POST/PATCH/DELETE | `/api/admin/activities/:id?` | 活动字段 | 活动管理 |
| POST/PATCH/DELETE | `/api/admin/posts/:id?` | 帖子字段 | 社区帖子管理 |
| PATCH/DELETE | `/api/admin/posts/:postId/comments/:commentId` | `status` | 评论审核或删除 |
| POST/PATCH/DELETE | `/api/admin/notices/:id?` | 公告字段 | 内容管理 |

后台商品新增示例：

```json
{
  "name": "咖啡书签",
  "description": "文创商品",
  "price": 18.8,
  "stock": 20,
  "category": "creative"
}
```

后台支付审核示例：

```json
{
  "status": "approved"
}
```

## 错误处理说明

- 后端统一使用 `ok()` 和 `fail()` 输出响应。
- 参数不合法返回 400。
- 未登录返回 401。
- 越权访问返回 403。
- 资源不存在返回 404。
- 库存不足、重复报名、重复点赞等业务冲突返回 409。
- 未捕获服务端异常会记录到后端终端，返回用户友好的 500 文案。
