# 接口说明

接口基础地址默认是 `http://localhost:4173`，返回格式统一为：

```json
{
  "success": true,
  "data": {}
}
```

失败时返回：

```json
{
  "success": false,
  "message": "错误说明"
}
```

## 登录鉴权

- 普通用户登录后使用 `Authorization: Bearer <user-token>`。
- 管理员登录后使用 `Authorization: Bearer <admin-token>`。
- 后端会校验 token 类型，普通用户 token 不能访问管理员接口。

## 安全规则

- 后台接口统一在路由入口处校验 `currentAdmin(req)`。
- 用户私有数据通过 `currentUser(req)` 获取当前用户。
- 订单支付状态和支付提交会校验 `order.userId === user.id`。
- 预约取消会校验 `reservation.userId === user.id`。
- MySQL 写入和删除使用 `pool.execute` 参数化语句，避免把用户输入拼接到 SQL 值中。
- 前端 Vue 模板默认转义插值文本，当前 Vue 页面未使用 `v-html` 展示用户输入。
