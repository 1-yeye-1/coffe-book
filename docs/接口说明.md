# 接口说明

后端服务默认地址：

```text
http://localhost:4173
```

接口返回结构兼容：

```json
{
  "success": true,
  "code": 200,
  "msg": "ok",
  "data": {}
}
```

## 公开接口

| 方法 | 路径 | 说明 |
| --- | --- | --- |
| GET | `/api` | 接口索引。 |
| GET | `/api/home` | 首页聚合数据。 |
| GET | `/api/products` | 商品列表，支持分类和关键词。 |
| GET | `/api/products/:id` | 商品详情。 |
| GET | `/api/books` | 图书列表。 |
| GET | `/api/books/:id` | 图书详情。 |
| GET | `/api/activities` | 活动列表。 |
| GET | `/api/activities/:id` | 活动详情。 |
| GET | `/api/posts` | 社区帖子列表。 |
| GET | `/api/seats/status` | 座位状态。 |
| GET | `/api/qr?data=...` | 生成支付或礼券二维码。 |

## 安全约束

- 前台会员接口需要用户 token。
- 后台接口需要管理员 token。
- 订单、预约、支付等接口均校验资源归属。
- 注册、登录、资料更新、购物车、订单、预约、活动和社区接口均有基础参数校验。
